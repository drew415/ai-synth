# AI Synth

A production-quality polyphonic subtractive synthesizer built with Web Audio API.

## Features

- Polyphonic voice architecture with configurable voice count
- Dual oscillators with multiple waveforms
- Sub oscillator and noise generator
- Resonant lowpass/highpass/bandpass filter
- Dual ADSR envelopes (amp + filter)
- Unison with detune and stereo spread
- LFO modulation
- FX chain: Distortion, Delay, Reverb, Chorus
- MIDI input support
- Preset management

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Web Audio API
- Web MIDI API

## Development

```bash
npm install
npm run dev
```
